# YouTube Video Details Fetcher

This script fetches details of a YouTube video using the YouTube Data API v3.

## Setup Instructions

### Prerequisites
- Python 3.x
- `pyyoutube` library

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your_username/your_repository.git
   cd your_repository

### Make sure to:
